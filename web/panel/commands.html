Comandos

(sender) Usuario que utiliza el comando.
(@sender) Usuario que utiliza el comando y añade el simbolo @ a su nombre.
(baresender) Usuario que utiliza el comando en minusculas.
(touser) Remplaza el argumento que se escribe despues del comando. Si no hay nada despues, se utiliza el nombre del usuario que lo utiliza.
(#) Numero aleatorio.
(1), (2), (3) etc Argumentos del comando.
(age) Dice desde cuando estas en Twitch.
(random) Persona aleatoria del chat.
(pointname) Nombre de los puntos actual.
(uptime) Tiempo que se lleva en directo.
(game) Juego puesto en Twich actualmente.
(status) Titulo puesto en Twitch actualmente.
(price) Precio del comando.
(pay) Pago del comando.
(onlineonly) Hace que el comando solo se utilice cuando el directo esta encendido.
(offlineonly) Hace que el comando solo se utilice cuando el directo esta apagado.
(viewers) Contador de espectadores actual.
(count) Cantidad de veces que se utilizo el comando.
(echo) Permite decir cualquier mensaje como si fuese el bot.
(1=) Argumento 1. Añade al nombre de etiqueta de tu preferencia despues de = si el argumento esta vacio.
(countdown=) Añade una fecha que quieras contar regresivamente. Ej (countdown=July 2 2016 7:00).
(downtime) Tiempo desde el ultimo directo.
(readfile filename.txt) Lee que hay en el archivo. El archivo debe estar en la carpeta addons del bot.
(customapijson URL key | {cadena literal} ...) Llama a una URL remota que retorna un JSON. Llave u objeto.Llave u objeto.objeto.Llave para acceder a los datos del JSON.
(command [nombre_del_comando]) Llama un comando. Debe ser el primer objeto en el comando personalizado. No incluya ! antes que el nombre_del_comando
(customapi [requesturl]) Llama a una URL remota.
